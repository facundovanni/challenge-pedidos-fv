# Pedidos Fast

Este proyecto ha sido desarrollado utilizando **Angular 18** con componentes **Standalone** y estilos implementados nativamente con **Tailwind CSS**, priorizando el rendimiento, la mantenibilidad y un diseño limpio.

## 🚀 Instalación y Ejecución

1. **Clonar el repositorio:**
  ```bash
   git clone <url-de-tu-repo>
   cd challenge-pedidos-fv
  ```

2. **Instalar dependencias**
  ```bash
   npm install
  ```

3. **Instalar dependencias**
  ```bash
   npm start
  ```
  La aplicación estará disponible en http://localhost:4200.

##  Ejecución de Tests Unitarios

  DEV
  ```bash
  npm test
  ```
  
  CI
  ```bash
  npm run test:ci
  ```

##  Build

  ```
  npm run build
  ```
